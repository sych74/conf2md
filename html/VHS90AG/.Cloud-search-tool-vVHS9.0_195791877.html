Virtuozzo Hybrid Server 9.0 Admin Guide : .Cloud search tool vVHS9.0

Created by Roksoliana Ben, last modified on May 02, 2024

The search tool in the upper left corner allows you to search your cloud for:

  • IP addresses
  • Usernames
  • Users full names (first or last name, in any order)
  • VM labels
  • VM identifiers
  • Disk identifiers
  • Log IDsÂ 
  • Backups
  • Template labels
  • Permission labels/identifiers
  • OnApp page URLs/titles
  • Any word from locale texts (yellow help box texts), if other search results are not successful

Type what you want to search for into the search box and confirm.
The results are organized according to the menu item they refer to. For example, Pages, Virtual Machines, Users, Locales, etc. Click a search result to open the relevant details page.
