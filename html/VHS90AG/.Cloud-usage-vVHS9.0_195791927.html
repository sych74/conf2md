Virtuozzo Hybrid Server 9.0 Admin Guide : .Cloud usage vVHS9.0

Created by Roksoliana Ben, last modified on May 02, 2024

To view the usage statistics of your cloud, go to your Control Panel > Metrics > Cloud Usage.  

Usage statistics details

The Usage Statistics page lists each virtual machine in your cloud, along with the following details:

  • Virtual Server - the label of the virtual machine.
  • Owner - the username of the VM owner.
  • CPU Used - the average CPU percentage that the VM has been using during the last 48 hours or during the period you specified.
  • Disk r/w Completed
    • Disk reads completed - the number of read operations completed on the disk.
    • Disk writes completed - the number of write operations completed on the disk.
  • Disk r/w Data
    • Disk data read - the amount of data read from the disk.
    • Disk data written - the amount of data written to the disk.
  • Bandwidth 
    • Bandwidth sent - the number of Kilobytes (KB) sent by the VM.
    • Bandwidth received - the number of Kilobytes (KB) received by the VM.

Sort statistics

To sort the statistics in the table, hover over the required column and click the label. The arrow that appears while hovering over the label shows the order (ascending/descending) in which the data is sorted. Click the label again to sort the data in a different order. You can also view details of a specific VM or its owner by clicking the corresponding links in the table. 

Statistics period

By default, statistics are generated for the last 48 hours. To specify another period, set the Start and End times and click the Apply button.

When you specify hours for statistics generation, pay attention that statistics will be generated as follows:

  • For a period where hours are specified without minutes, for example, 15.00-17.00, the statistics will be generated for the specified period and the preceding hour, that is from 14.00 up to 17.00.
  • For a period where hours are specified with minutes, for example, 15.30-17.00, the statistics will be generated for the specified period, which is from 15.00 up to 17.00.
  • To generate the statistics for one month, select a period without the following month, for example, from 01.04.2019 00:00 to 30.04.2019 23:59 but not to 01.05.2019 00:00. 

Download statistics

To download a CSV file with statistics for the selected period, click the Save as CSV button. The download will start automatically after you click the button. In the downloaded file, the data in the data_sent, data_received, data_read, and data_written columns are displayed in GB.
