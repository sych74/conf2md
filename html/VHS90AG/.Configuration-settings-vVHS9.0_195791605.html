The purpose of a user service account is to perform actions inside a virtual machine, such as configuring an IP, disk, etc. If the service account is disabled or deleted, all operations related to the OS do not work.

The configuration settings screen lets you change various aspects of your OnApp installation. To edit these OnApp configuration settings:

  1. Go to your Control Panel > Admin > Settings menu.
  2. Click the Configuration icon. Click the tabs to edit the relevant config settings (System, Backups/Templates, Interface, Defaults, or Infrastructure)—these are explained in the following subsections:

  • Editing system configuration
  • Editing backups/templates configuration
  • Editing interface configuration
  • Editing defaults configuration
  • Editing infrastructure configuration
