Virtuozzo Hybrid Server 9.0 Admin Guide : .Control Panel configuration vVHS9.0

Created by Roksoliana Ben, last modified on May 02, 2024

You get detailed control over the configuration of Control Panel in the OnApp Configuration menu accessed at Control Panel > Admin > Settings menu. The menu includes details and settings for the following items:

  • Hardware info
  • License
  • Configuration settings
  • Look & feel
  • Localization and customization
  • Global whitelist
  • Service Insertion Framework configuration
  • Managing languages
