Virtuozzo Hybrid Server 9.0 Admin Guide : .Data store zones settings vVHS9.0

Created by Roksoliana Ben, last modified on May 07, 2024

Data store zones can be used to create different service tiers – for example, by setting up different zones for ordinary and high-performance SANs in the cloud. 

Data store zones have types inherited by the data stores in the zone. Later, data stores can be attached to a compute resource/compute zone of the same type. Data stores can be moved from one data store zone to another, but the zones should have the same type. For more information, refer to Zone types. The table below demonstrates the available data store zone types for different data stores:

Ignore the options specified in your Control Panel but not mentioned in the documents.

Create Data Store Zone

  1. Go to your Control Panel > Admin > Settings menu and click the Data Store Zones icon.
  2. Click the Create Data Store Zone button.
  3. On the screen that follows:
    • Label - give your data store zone a name.
    • Server type - choose the server type from the drop-down list:
      • Choose the virtual server type to create a KVM zone.

    The zone's type cannot be changed after the zone is created.

  4. Click Save.

Edit Data Store Zone

  1. Go to your Control Panel > Admin > Settings menu, and click the Data Store Zones icon. 
  2. To change the zone's name, click the Actions button next to the data store zone you are interested in, then select Edit. 
  3. On the page that loads, you can change the following data tore zone details:
    • Label - give your data store zone a name.
  4. Click Save.

Attach Data Stores to Data Store Zone

  1. Go to your Control Panel > Admin > Settings menu and click the Data Store Zones icon.
  2. Click the label of the zone you want to add a data store to.

  3. On the following screen, click next to the unassigned data store you want to add.

    When you add a data store to a data store zone, the data store inherits the zone's type. For more information, refer to Zone types.

Remove Data Stores from Data Store Zone

  1. Go to your Control Panel > Admin > Settings menu and click the Data Store Zones icon.
  2. Click the label of the zone you want to remove the data store from.
  3. On the screen that appears, click  next to the data store you want to remove.

Delete Data Store Zone

  1. Go to your Control Panel > Admin > Settings menu, and click the Data Store Zones icon. 
  2. Click the Actions button next to the zone you want to remove, then click Delete. 
  3. Confirm the deletion.
