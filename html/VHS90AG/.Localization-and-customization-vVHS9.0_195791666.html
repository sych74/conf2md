The search box in the Localization and Customization menu allows you to search by the following parameters:

  • Item ID
  • English Value
  • Translation

Search Localization & Customization Menu

To search the Localization and Customization menu:

  1. Go to your Control Panel > Admin > Settings menu.
  2. Click the i18n Customization icon.
  3. Click the required language Name (for example, "English").
  4. Type the search phrase into the search box and click Search.
  5. If required, make changes and click Apply.
