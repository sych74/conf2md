Virtuozzo Hybrid Server 9.0 Admin Guide : .Managing compute zone data stores vVHS9.0

Created by Roksoliana Ben, last modified on May 02, 2024

Data stores can easily be attached and removed from compute zones. This association between a compute zone and a data store is called a data store join.

Ignore the options specified in your Control Panel but not mentioned in the documents.

You can add data stores to a compute zone only if the data store belongs to a data store zone of the same type as the compute zone. For more information, refer to Zone types.
  1. Go to your Control Panel > Admin > Settings menu.
  2. Click the Compute Zones icon.
  3. Click the label of the compute zone you want to manage data stores for.
  4. Click the Tools button, and then select Manage Data Stores.
  5. The following screen shows a list of all data stores currently associated with this compute zone.

To remove a data store from the zone, click  next to it. You'll be asked for confirmation before the store is removed.

To add a new data store, choose one from the drop-down list and click the Add Data Store button.
