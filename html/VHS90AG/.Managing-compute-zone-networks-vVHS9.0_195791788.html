Virtuozzo Hybrid Server 9.0 Admin Guide : .Managing compute zone networks vVHS9.0

Created by Roksoliana Ben, last modified on May 02, 2024

Networks can easily be attached and removed from compute zones. This association between a compute zone and a network is called a network join.

Ignore the options specified in your Control Panel but not mentioned in the documents.

  1. Go to your Control Panel > Admin > Settings menu.
  2. Click the Compute Zones icon.
  3. Click the label of the compute zone you want to manage networks for.
  4. Click the Tools button, and then select Manage Networks.
  5. On the following screen, you'll see a list of all networks currently associated with this compute zone.

To remove a network, click  next to it. You'll be asked for confirmation before the network is removed.

To add a new network, choose one from the drop-down list, enter its interface name (eth0, eth1), and click the Add Network button.

  • If the compute zone loses network connection, a warning message All compute resources are not responsive will be displayed.
  • You can add networks to a compute zone only if the network belongs to a network zone of the same type as the compute zone. For more information, refer to Zone types.
