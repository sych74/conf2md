Virtuozzo Hybrid Server 9.0 Admin Guide : .Managing failover vVHS9.0

Created by Roksoliana Ben, last modified on May 02, 2024

Compute resource failover means VM migration to another compute resource if the compute resource on which it is running goes offline.

Ignore the options specified in your Control Panel but not mentioned in the documents.

If you want to enable/disable failover for all compute resources within the compute zone, do the following:

  1. Go to your Control Panel > Admin > Settings menu.
  2. Click the Compute Zones icon.
  3. Click the label of the compute zone you want to manage failover for.
  4. Click the Tools button, and then choose Manage Failover.
  5. Click the Enable All button to enable failover for all compute resources within this zone. 
  6. If the failover is already enabled you can disable it by clicking the Disable All button.

When you change the compute zone's failover status, this change will be applied to all compute resources within this zone.
