Virtuozzo Hybrid Server 9.0 Admin Guide : .Managing suspended virtual machines vVHS9.0

Created by Roksoliana Ben, last modified on May 02, 2024
For information on how to suspend a virtual machine, refer to the Virtual Machine Power Options section.

Suspending a virtual server makes it inactive, but still present on the system. All the major actions on the suspended VM are disabled except for some actions related to:

  • Properties
  • CPU Usage 
  • Billing Statistics
  • Network Interfaces
  • Firewall Rules
  • Disks
  • Backups
To run all the available VM-related actions on a suspended instance, you should unsuspend it. 

Ignore the options specified in your Control Panel but not mentioned in the documents.

Properties

You can view the general properties of the suspended VM at Control Panel > Cloud > Virtual Servers menu, including such details as hostname, compute resource, owner, IP addresses, and others. You can also view the Notes section that lists brief comments or reminders for the suspended VM but you cannot create new notes. 

For more information on the VM properties, refer to the Virtual Machine Details section.

CPU Usage

You can view charts on the CPU usage of the suspended VM at Control Panel > Cloud > Virtual Servers > Label > Overview > CPU Usage tab. The charts show the total CPU usage statistics for all the cores of the particular VM for a specified time period. The vertical axis indicates the CPU usage percentage (CPU percentage is the core-independent quantity) and the horizontal axis defines a time period. 

For more information on CPU usage, refer to the Virtual Machine CPU Utilization section.

Billing Statistics

You can view billing statistics of the suspended VM at Control Panel > Cloud > Virtual Servers > Label > Overview > Billing Statistics tab. The Billing Statistics page contains a record of billing operations relevant to the last three-month period. If a virtual machine was created less than three months ago, statistics are recorded starting from the VM's creation date. You can view all available statistics or those for a shorter period by setting a Start and End time.

For more information on how to view billing statistics, refer to the Virtual Machine Billing Statistics page.

Network Interfaces

You can view the virtual network interfaces allocated to the suspended VM at Control Panel > Cloud > Virtual Servers > Label > Networking > Network Interfaces tab, including details about the network join, port speed, and primary network interface. For other operations related to managing network interfaces, you need to unsuspend your virtual machine. 

For more information on managing network interfaces, refer to the Virtual Machine Network Interface section.

Firewall Rules

You can view firewall rules that were added to VM before its suspension at Control Panel > Cloud > Virtual Servers > Label > Networking > Firewall tab. You can also remove a firewall rule from the suspended VM. For other operations related to firewall settings, you need to unsuspend your virtual machine.

For more information on firewall rules, refer to the Setting Virtual Machine Firewall Rules page.

Disks

You can view the list of disks allocated to the suspended VM at Control Panel > Cloud > Virtual Servers > Label > Storage > Disks tab. For other operations related to disk settings, you need to unsuspend your virtual machine.

For more information on how to manage disks, refer to the Virtual Machine Disks section.

Backups

You can view the suspended VM backups that have already been taken and that are scheduled to be taken at Control Panel > Cloud > Virtual Servers > Label > Backups tab, take the backup, and convert it to a template. For other operations related to managing backups, you need to unsuspend your virtual machine.

For more information on the VM backups, refer to the Virtual Machine Backups page.
