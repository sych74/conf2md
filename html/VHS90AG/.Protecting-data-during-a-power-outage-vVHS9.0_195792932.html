Virtuozzo Hybrid Server 9.0 Admin Guide : .Protecting data during a power outage vVHS9.0

Created by Roksoliana Ben, last modified on May 07, 2024

To protect Virtuozzo Hybrid Server against power outages, it is recommended to use an Uninterruptable Power Supply (UPS) for all servers and network switches.

Additionally, you can prevent data loss and preserve data integrity using enterprise-grade SSD drives. Unlike HDD drives, enterprise-grade SSD drives can properly handle power loss events. Enterprise-grade SSD drives that operate correctly usually have the power loss protection property in their technical specification. Some of the market names for this technology are Enhanced Power Loss Data Protection (Intel), Cache Power Protection (Samsung), Power-Failure Support (Kingston), and Complete Power Fail Protection (OCZ).

It is also recommended to ensure that all storage devices you will add to your cluster can flush data from the cache to the disk in case of a power outage. You can check the data flushing capabilities of your disks, as explained in Checking disk data flushing capabilities.
