Created by Roksoliana Ben, last modified on May 02, 2024

CPU quota is a percentage value limiting maximum VM CPU load on a compute resource. CPU quota functionality allows limiting CPU usage for the particular virtual machine order to avoid abuse usage, which is affecting all virtual machines on the KVM compute resource. 

  • This option is available for the users under administrator's role. Make sure you have enabled the Manage CPU quota permission first.
  • This feature is available only for KVM compute resources.
  • Before you enable CPU quota, its value is set to unlimited for all the VMs on this compute resource.


You can set the default value of CPU quota on the compute resource level and edit the custom value on the virtual machine level.


Set CPU Quota for Compute Resource

  1. Go to your Control Panel > Admin > Settings menu.
  2. Click the Compute Resources icon.
  3. Click the label of the compute resource you are interested in.
  4. On the screen that appears, click the Actions button, point to Resource Options, and select Set default CPU Quota.
  5. Move the CPU Quota enabled slider to the right to enable CPU quota and set the default value.

  6. Set CPU quota. The maximum value is 99%. Also, you can select the ∞ unlimited checkbox to set an unlimited amount of CPU quota.  

  7. Click the Save button.

  • If the default CPU quota value is changed or the CPU quota is enabled, it does not affect running virtual machines until they are restarted.
  • If the default CPU quota is disabled, it is set unlimited for all running virtual machines.
