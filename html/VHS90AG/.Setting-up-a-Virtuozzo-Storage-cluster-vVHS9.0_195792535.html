Virtuozzo Storage is a software-defined storage solution that lets you quickly and easily transform commodity hardware and network equipment into protected enterprise-grade storage like storage area networks (SAN) or network-attached storage (NAS).

Virtuozzo Storage is optimized for storing large amounts of data and provides data redundancy (replication and erasure coding) and self-healing.

This chapter provides information on setting up a Virtuozzo Storage cluster. It starts with an overview of the Virtuozzo Storage setup and then describes each setup step in detail.

Do not run the console commands described in this chapter if you have the Virtuozzo Storage management panel installed. Doing so may break your Virtuozzo Storage installation. Instead, perform the same tasks via the management panel, as explained in the Virtuozzo Storage Administrator’s Guide.
