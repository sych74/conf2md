  • Even though a cluster can be created on top of varied hardware using nodes with similar hardware in each node will yield better cluster performance, capacity, and overall balance.
  • We highly recommend using UEFI instead of BIOS if it is supported by your hardware. It is recommended particularly if you use NVMe drives.
  • Any cluster infrastructure must be tested extensively before deploying it to production. Such common points of failure as SSD drives and network adapter bonds must always be thoroughly verified.
